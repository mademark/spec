## Overview

Made Mark answers the question: **How was this content made?**

Three marks define authorship clearly:

1. **Human Made** — Conceived and produced by people
2. **Human Designed, AI Made** — Human creative direction, AI production
3. **AI Made** — Conceived and produced primarily by AI

Full documentation at [mademark.org](https://mademark.org)

## Machine Layer

Each mark resolves as JSON-LD at:
- `mademark.org/marks/human-made`
- `mademark.org/marks/human-designed-ai-made`
- `mademark.org/marks/ai-made`

Namespace: `mademark.org/ns`

Compatible with C2PA content authentication manifests.
